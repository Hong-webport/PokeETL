Instructions

Search on what you want in the correct search bar. Then clicked on the circle that appeared more info. If you do multiple or begin a new search, You might need to scroll down to see it. See what moves your POKEMON can learn.

Ability: Passive effect in battle or in the overworld. Individual Pokémon may have only one Ability at a time.

Click on the middle node to see what Pokemon can attain it.

Technique/Move: also known as an ATTACK, is the skill Pokémon primarily use in battle.

Pokemon Type:

The technique belong to one type and determine its effectiveness.

Stats:

Accuracy: The percent value of how likely this move is to be successful.
Power: The base power of this move with a value of 0 if it does not have a base power.
Use Limits: Power Points. The number of times this move can be used.
Effect Chance: the percentage of the likelihood of the effect occurrence.
